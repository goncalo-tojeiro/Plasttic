## Parcel Workflow

SETUP:

- Node.js install (https://nodejs.org/)
- npm -v (NPM version check)
- npm init -y (Initialize NPM Package)
- package.json (edit Package Info)
- npm install parcel-bundler --D (Parcel install)
- npm install sass --D
